# 🎨 Intro POO Canvas 🚀

¡Bienvenido! Este proyecto es una exploración divertida de la **Programación Orientada a Objetos (POO)** aplicada al dibujo y animación con el elemento `<canvas>` de HTML5. 📜✨

---

## ✨ Características Principales

* **🏗️ Estructura POO**: Uso de la clase `Circle` para crear objetos con vida propia.
* **🛡️ Bordes Seguros**: Lógica matemática para que los círculos nunca se escapen del canvas.
* **🌈 Explosión de Color**: Generación dinámica de colores RGB aleatorios.
* **♾️ Animación Infinita**: Bucle fluido utilizando `requestAnimationFrame`.

---

## 🛠️ Tecnologías Utilizadas

* **🌐 HTML5 (Canvas)**: El lienzo donde ocurre la magia.
* **⚡ JavaScript (ES6+)**: El cerebro detrás de los objetos y animaciones.
* **🎨 CSS3**: Para darle estilo a los contenedores.
* **🐙 Git/GitHub**: Control de versiones y despliegue.
